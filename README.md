## Подстановка секретных переменных в окружение
BOT_TOKEN - токен вашего бота 

POSTGRES_DB - название базы данных

POSTGRES_USER - имя пользователя 

POSTGRES_PASSWORD - пароль 

POSTGRES_HOST - хост
